## 🎯 O que você vai aprender

## 💡 Dica de ouro

Commit **cedo**, commit **sempre**. Mensagem clara. O seu eu do futuro agradece — especialmente às 2h da manhã tentando lembrar o que você fez há 3 semanas.

```bash
# ❌ Não faça isso
git commit -m "ajustes"
git commit -m "mais ajustes"
git commit -m "agora vai"

# ✅ Faça isso
git commit -m "fix: corrige validação do formulário de login"
git commit -m "feat: adiciona botão de recuperar senha"
git commit -m "docs: atualiza README com instruções de setup"
```
